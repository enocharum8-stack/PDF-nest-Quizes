Library Concepts: The word library derives from the Latin liber (book). Ranganathan's Five Laws: Books are for Use · Every Reader His Book · Every Book Its Reader · Save the Time of the User · The Library is a Growing Organism. Types of libraries: National, Public, School, Academic, Special.

ICT & Computer: ICT = information gathering, processing, storage, retrieval & dissemination. Computer generations: 1st (vacuum tubes) → 2nd (transistors) → 3rd (ICs) → 4th (microprocessors) → 5th (AI). Hardware (input/processing/output/storage/communication) vs Software (system/application).

Library Automation: Use of ICT tools to make library services efficient. Examples: KOHA, Evergreen, DSpace, Greenstone. Proprietary vs Open-source software. MARC & Dublin Core = metadata standards. Evolution: Traditional → Automated → Digital → Virtual.

Copyright & Plagiarism: Copyright (WIPO 2014) = legal rights creators have over literary/artistic works. Fair Use = limited reproduction with no effect on original value. Plagiarism = presenting another's work as your own. Piracy = illegal reproduction for commercial gain.
